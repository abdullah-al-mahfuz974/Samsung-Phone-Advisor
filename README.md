#Samsung Phone Advisor
- A smart assistant for Samsung smartphone recommendations.
- Stores phone specifications in **PostgreSQL**.
- Implements a **RAG module** to retrieve structured data.
- Uses a **multi-agent system** to generate natural-language reviews and comparisons.
- Unified `ask(question)` function simulates a FastAPI endpoint.
- Users can ask questions like:
  - “Compare Galaxy S23 Ultra and Galaxy S22 Ultra”
  - “Which Samsung phone has the best battery under $1000?”
  - “What are the specs of Galaxy A54?”
